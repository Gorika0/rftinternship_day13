# rftinternship_day13

This project demonstrates Distribution Analysis using Python and Seaborn.
A histogram is created using student marks data to understand data distribution visually.
The project also includes a KDE (Kernel Density Estimation) curve and skewness analysis.

pandas → Used for data handling and creating tables.
seaborn → Used for creating statistical graphs.
matplotlib.pyplot → Used for displaying plots and charts.
histplot() → Creates histogram graph.
bins=8 → Divides data into 8 groups.
kde=True → Adds KDE curve for smooth distribution analysis.
Skewness Meaning
Near 0 → Normal distribution
Positive value → Right-skewed distribution
Negative value → Left-skewed distribution
